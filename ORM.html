Class: ORM

ORM()

ORM - the Object Relational Mapper.

Use instances of this class to:

  • Register your Model classes using ORM#register
  • Get the empty state for the underlying database with ORM#getEmptyState
  • Start an immutable database session with ORM#session
  • Start a mutating database session with ORM#mutableSession

Internally, this class handles generating a schema specification from models to the database.
